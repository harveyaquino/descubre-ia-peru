# IA Perú - Taller de Inteligencia Artificial

Plataforma web para capturar leads y gestionar participantes del taller de IA para empresas peruanas. Construida con Next.js, Supabase y Vercel.

## 🚀 Stack Tecnológico

- **Frontend**: Next.js 14 + React 18
- **Backend**: Supabase (PostgreSQL)
- **Hosting**: Vercel
- **Versionado**: GitHub

## 📋 Configuración Inicial

### 1. Clonar el Repositorio

```bash
git clone https://github.com/tu-usuario/ia-peru-taller.git
cd ia-peru-taller
```

### 2. Instalar Dependencias

```bash
npm install
```

### 3. Configurar Supabase

1. Crear una cuenta en [Supabase](https://supabase.com)
2. Crear un nuevo proyecto
3. En la sección **SQL Editor**, ejecutar:

```sql
CREATE TABLE leads (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  name TEXT NOT NULL,
  email TEXT NOT NULL,
  company TEXT NOT NULL,
  industry TEXT,
  challenge TEXT,
  phone TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Crear índice para queries rápidas
CREATE INDEX idx_leads_email ON leads(email);
CREATE INDEX idx_leads_created_at ON leads(created_at);
```

4. Obtener tus credenciales:
   - Ir a **Project Settings** > **API**
   - Copiar `Project URL` y `anon public key`

### 4. Configurar Variables de Entorno

Crear archivo `.env.local`:

```bash
NEXT_PUBLIC_SUPABASE_URL=tu_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=tu_anon_key
```

### 5. Ejecutar en Desarrollo

```bash
npm run dev
```

Visitar `http://localhost:3000`

## 🌍 Desplegar en Vercel

1. Subir código a GitHub
2. Ir a [Vercel](https://vercel.com)
3. Importar el repositorio
4. Añadir variables de entorno
5. Desplegar

2. Seleccionar tabla `leads`
